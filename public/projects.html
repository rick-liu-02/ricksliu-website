AnimeList Duel

HTML, CSS, React, Node.js, MySQL

The premise was simple: Who's wasted more time watching Japanese cartoons?

The solution? A website that pits your anime-watching stats head-to-head with another user by pulling data from the website MyAnimeList.

The frontend uses HTML, CSS and React, and the backend uses Node.js and MySQL. Inputted usernames are sent to a backend that scrapes data from various MyAnimeList pages and sends it back. This data is also stored in a MySQL database so future queries for the same user are much faster. The website still functions without the database though, just less ideally.

CamWow!

Python

What if you could set up automated tasks for your computer based on hand gestures?

'CamWow!' was a first-year school project made by a team of five. A Raspberry Pi recognizes hand gestures, including various shapes and the number of fingers, and sends this data to a computer (or multiple). Based on what the computer gets, certain user-specified tasks are automatically run. These actions can be opening computer files or opening links with optional login info, and a GUI makes customizing these things easier. The entire thing was made in Python, the key libraries being OpenCV, Selenium and Tkinter.

Of course, I only worked on 20% of this. Specifically, I was responsible for establishing communications between the Pi and the computer, implementing the ability to customize and save tasks, and creating the GUI.

Still confused? Unfortunately, the source code can never see the light of day, but check out the definitely unbiased video showcasing the project.

MyAnimeList Score Neural Network

Python

How well can a neural network predict user ratings on MyAnimeList?

Quite poorly, as it turns out. The network was built with Keras, the high-level Tensorflow API. To pull data from MyAnimeList, I used a community-made API called JikanPy. On average, it's around 0.9 points off out of 10, but sometimes it's way off. I guess that's better than guessing randomly.

RickBot

Python

A Discord bot I made for a private Discord server. The only real feature is the ability to generate fake messages based on specific users. It does this by analyzing previous chat logs and generating new text with a Markov chain.
